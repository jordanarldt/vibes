## Usage

<CodeBlock lang="bash">{`import { Cart } from '@/vibes/soul/sections/cart'`}</CodeBlock>

<CodeBlock lang="bash">{`const products = await getProducts()
const subtotal = await getSubtotal()`}</CodeBlock>

<CodeBlock lang="bash">{`<Cart
  title="Cart"
  lineItems={products}
  summary={{
    title: 'Summary',
    caption: 'Shipping & taxes calculated at checkout',
    subtotal: subtotal,
    subtotalLabel: 'Subtotal',
    shippingLabel: 'Shipping',
    taxLabel: 'Tax',
    tax: '$11.60',
    grandTotalLabel: 'Total',
    grandTotal: '$127.60',
    ctaLabel: 'Checkout',
  }}
  emptyState={{
    title: 'Your cart is empty',
    subtitle: 'Add some products to get started.',
    cta: {
      label: 'Continue shopping',
      href: '#',
    },
  }}
  removeLineItemAction={removeLineItemAction}
  updateLineItemQuantityAction={updateLineItemQuantityAction}
  // redirectToCheckoutAction={}
/>
`}</CodeBlock>

### Internationalization

Integrating internationalization into your Vibes components starts at the application level. You'll need a dictionary of strings that correspond to the properties of the `Cart` component. For example, the `title` property of the `Cart` component would have a corresponding key-value pair in the dictionary like so:

<CodeBlock lang="json">
  {`
{
  "Cart": {
    "title": "Hello world!",
    //...
  }
}`}
</CodeBlock>

To setup internationalization, we recommend using a package like [next-intl](https://next-intl-docs.vercel.app/). After it's configured, you can retrieve translated strings via the `useTranslations` hook. You can then pass the translated string into each component property appropriately.

<CodeBlock lang="tsx">
  {`import { useTranslations } from 'next-intl';
 
export default function CartPage() {
  const t = useTranslations('CartPage');
  return (
    <div>
        <Cart
            title={t('title')}
            // ...
        />
    </div>
  );
}`}
</CodeBlock>

For full setup instructions, refer to the [next-intl getting started docs.](https://next-intl-docs.vercel.app/docs/getting-started)

## API Reference

### `Cart`

| Prop                           | Type                                                                                                    | Default  |
| ------------------------------ | ------------------------------------------------------------------------------------------------------- | -------- |
| `title?`                       | `string`                                                                                                | `"Cart"` |
| `lineItems`                    | [`CartLineItem[]`](#cartlineitem) \| [`Promise<CartLineItem[]`](#cartlineitem)>                         |          |
| `summary`                      | [`CartSummary`](#cartsummary)                                                                           |          |
| `emptyState`                   | [`CartEmptyState`](#cartemptystate)                                                                     |          |
| `removeLineItemAction`         | `function` <Tooltip content='(id: string) => Promise<void>' />                                          |          |
| `updateLineItemQuantityAction` | `function` <Tooltip content='({ id, quantity } : { id: string; quantity: number }) => Promise<void>' /> |          |
| `redirectToCheckoutAction`     | `function` <Tooltip content='() => Promise<void>' />                                                    |          |

### `CartLineItem`

| Prop       | Type                                                     | Default |
| ---------- | -------------------------------------------------------- | ------- |
| `id`       | `string`                                                 |         |
| `image`    | `Image` <Tooltip content="{src: string, alt: string}" /> |         |
| `title`    | `string`                                                 |         |
| `subtitle` | `string`                                                 |         |
| `quantity` | `number`                                                 |         |
| `price`    | `string`                                                 |         |
